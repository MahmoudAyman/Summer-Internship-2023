How to make AVR Programmer with ATtiny44A

Overview

This documentation provides a simplified guide to building an AVR programmer with an ATtiny44A microcontroller. Follow the steps carefully and refer to the provided references for additional information and guidance.

Components and Tools

  • ATtiny44A SMD microcontroller
  • 6-pin ISP header or 2x3 Pin Header SMD
  • 20 MHz Resonator
  • 330-ohm Resistor
  • LED (optional for status indication)
  • PCB board
  • Programmer (USBASP or Arduino as ISP)

    • Design Steps

    Step 1: PCB Design

    1. I used KiCad to design my PCB.
    2. Initially, I designed a USB connector package to minimize size. However, since I couldn't find a through-hole (THT) ATtiny84/44, I switched to using an SMD ATtiny44A and kept other components as THT.
    3. I tried to make my design single-layer , but due to spaces for a vertical 1x6 THT ISP header, I switched to a 2x3 SMD pin header to make it single single-layer

    Step 2: PCB Fabrication

    1. I used FlatCAM and an SRM20 machine to fabricate my PCB. With min trace 0.6 and min clearness 0.4

    Firmware Compilation (Linux)

    Step 1: Install Required Tools

      I installed the necessary tools on my Linux system:

    1. gcc-avr for code compilation.
    2. avr-libc, the C library for AVR.
    3. avrdude for uploading HEX files to the microcontroller.
      4.  sudo apt-get install gcc-avr avr-libc avrdude

    Step 2: Create a Makefile

    1. I created a Makefile for my project. Here's an example:
      MCU=ATtiny44A
  CC=avr-gcc
  OBJCOPY=avr-objcopy
  CFLAGS=-Wall -g -Os -mmcu=${MCU}
  TARGET=main
  SRCS=main.c
  
  all:
      ${CC} ${CFLAGS} -o ${TARGET}.bin ${SRCS}
      ${OBJCOPY} -j .text -j .data -O ihex ${TARGET}.bin ${TARGET}.hex
  
  flash:
      avrdude -p ${MCU} -c usbasp -U flash:w:${TARGET}.hex:i -F -P usb
  
  clean:
      rm -f *.bin *.hex

    Step 3: Compilation and Programming

    1. I used the Makefile to compile my code and generate executable files.
    2. I compiled the firmware by running make.
    3. I flashed the firmware to the ATtiny44A using my programmer. Example command: make flash.
